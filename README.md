# LegacyWebsite

This repository is for the website of LEGACY 6.0 DAV Pratap Vihar, Ghaziabad

## Setup Instructions (Windows)

### Download as ZIP and Extract

Follow these steps to download the repository as a ZIP file and run the webpage locally on Windows:

1. **Download the Repository**
   - Visit the repository on GitHub
   - Click the green "Code" button
   - Select "Download ZIP"
   - The file will be saved as `LegacyWebsite-main.zip`

2. **Extract the ZIP File**
   - Locate the downloaded `LegacyWebsite-main.zip` file (usually in your Downloads folder)
   - Right-click on the ZIP file
   - Select "Extract All..." from the context menu
   - Choose a destination folder where you want to extract the files
   - Click "Extract"
   - A folder named `LegacyWebsite-main` will be created

3. **Run the Webpage Locally**
   - Navigate to the extracted folder (`LegacyWebsite-main`)
   - Open the `src` folder
   - Right-click on `index.html`
   - Select "Open with" and choose your preferred web browser (e.g., Chrome, Firefox, Edge)
   - The webpage will open in your default or selected browser

   Alternatively:
   - Open your web browser
   - Press `Ctrl+O` (or go to File > Open)
   - Navigate to the extracted folder and select `src/index.html`
   - Click "Open"

### Additional Notes

- No installation or server setup is required - the webpage runs entirely in your browser
- All other pages can be accessed by clicking on the navigation menu in `index.html`
